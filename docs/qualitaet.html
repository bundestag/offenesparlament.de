title: Hinweise zur Datenqualität Die von OffenesParlament.de verwendeten Daten entstammen der Webseite des Deutschen Bundestages. Die Webseite basiert auf einer maschinelle Auswertung der angebotenen HTML-Dokumente ("screen scraping"). Obwohl wir um eine korrekte Auswertung bemüht sind, ist dieses Vorgehen für Fehler anfällig. Wir möchten daher auch unsere Nutzer einladen, Code und Daten selbstständig gegenzuprüfen und uns auf Probleme hinzuweisen. * [Rohdatenzugriff](daten) zur Überprüfung der genutzten Datenbasis. * [Quellcode](https://github.com/pudo/offenesparlament) von OffenesParlament auf GitHub.
